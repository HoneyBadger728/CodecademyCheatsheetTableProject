Effectively creating tables using HTML and CSS

About

Creating tables in html is simply a means of organizing information into colums and rows. Presenting the information in this manner can allow the reader to more easily understand a set of data. Calendars, experiment results, schedules and tracking information are all exaples of data which is well suited to be presented in a table. This site will explain how to create various tables, provide examples, and detail the necessary steps (and information) for creating tables. This guide should serve as a point of reference for those seeking to create their own tables.

Tags

The first key piece of information, when it comes to creating tables, is becoming familiar with the fundamental <tags>. Below is a list of useful <tags>, along with a brief description. Examples are provided in the following section.

1.<table>

This element indicates that the data is to be represented in a series of rows and columns. This data will be defined within this element.

2.<caption>

The caption element allows for the creation of a title for the table. This will give context to the data being presented.

3.<thead>

This tag is used to set up the first row of the table. The information presented here will act as the headings for each of the various columns in the table.

4.<tbody>

Once the headings are defined, the rest of the data will go in the body of the table. This element separates the body from the headings.

5.<tfoot>

After the heading section and the body section, the foot section can be set aside to summarize the data presented in the table.

6.<tr>

This tag indicates a row within the table. It will act as the parent element for data within that row.

7.<th>

This element indicates that the data entered will act as a heading. It will either be placed within the <thead> element or as the first entry within a row.

8.<td>

This tag allows for the defining of raw data, within the table.
